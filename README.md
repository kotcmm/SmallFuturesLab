# SmallFuturesLab

SmallFuturesLab 是一个独立的小资金期货交易研究项目。

本项目只研究 1 万到 2 万资金规模下，是否存在可执行、可控制风险、可复盘的小资金期货交易方法。

本项目不继承任何其他项目的策略假设、市场结构定义、入场规则、止损规则、退出规则或参数体系。

---

## 1. 项目目标

SmallFuturesLab 的第一目标不是暴利，而是：

```text
生存
控制亏损
训练执行
积累真实交易样本
在不破坏账户的前提下寻找可持续优势
```

本项目研究的是：

```text
小资金账户在期货市场中，什么交易行为可以被账户承受；
哪些品种、周期、规则和风控方式可能适合 1 万到 2 万资金；
如何避免因为合约颗粒度、手续费、滑点、连续亏损导致账户快速损伤。
```

---

## 2. 项目不做什么

本项目明确不做：

```text
不追求短期暴利
不研究网格交易
不研究亏损摊平
不研究扛单回本
不研究重仓翻倍
不使用无法承受连续亏损的规则
不为了提高胜率而扩大亏损
不直接套用任何外部策略框架
```

---

## 3. 当前阶段

当前阶段只做三件事：

```text
1. 定义小资金账户约束
2. 建立品种与周期筛选框架
3. 形成候选策略方向，但暂不冻结具体交易规则
```

当前不写实盘代码，不接交易接口，不做收益率优化。

## 5. 研究纪律

任何候选策略必须先回答：

```text
一手最小风险是多少？
连续亏损能否承受？
手续费和滑点是否会吞噬优势？
是否需要隔夜？
是否会诱导重仓、摊平或扛单？
是否适合当前账户规模？
```

如果一个方法无法通过账户约束检查，则不进入策略研究。

---

## 6. 当前结论

SmallFuturesLab 从账户约束出发，而不是从既有策略概念出发。

本项目宁可推进慢，也不能在账户边界、品种边界、周期边界和风险边界尚未清楚时进入策略实现。
