# SmallFuturesLab

SmallFuturesLab 是一个独立的小资金期货生存研究项目。

本项目服务的账户规模暂定为：

```text
10,000 元到 20,000 元人民币权益。
```

本项目不从既有策略出发，也不从收益目标出发。

本项目只研究一个问题：

```text
1 万到 2 万小资金账户，如何不被合约颗粒度、杠杆、手续费、滑点、连续亏损和主观冲动淘汰，并在此基础上寻找可执行的小规模交易方法。
```

---

## 1. 项目第一目标

SmallFuturesLab 的第一目标是：

```text
生存。
```

优先级如下：

```text
账户生存 > 风险可解释 > 执行一致性 > 复盘能力 > 收益率
```

本项目不追求快速翻倍，不以任何冠军案例、收益率案例或单个交易者经历作为收益目标。

外部案例只能用于提取：

```text
风险约束；
执行纪律；
品种筛选思路；
不可承受行为；
交易心理警示。
```

外部案例不能用于复制收益率、复制仓位、复制交易频率或复制主观经验。

---

## 2. 核心原则

SmallFuturesLab 的核心原则是：

```text
小资金策略不是大资金策略的缩小版。

本项目不追求预测正确，
而是追求错误可承受。

如果一个交易行为在 1 万到 2 万账户下不可承受，
即使它可能赚钱，也不进入本项目当前阶段。
```

---

## 3. 当前阶段只解决什么

当前阶段只解决：

```text
1. 账户能承受什么风险；
2. 哪些交易行为必须禁止；
3. 哪些品种可能适合小资金继续研究；
4. 哪些周期可能适合人工执行和复盘；
5. 候选方向应该用什么标准评价。
```

当前阶段不解决：

```text
最终策略规则；
最终入场信号；
最终止损规则；
最终收益目标；
自动化交易；
实盘交易。
```

---

## 4. 当前阶段明确不研究

当前阶段明确不研究：

```text
高频盘口交易；
每天数十笔交易；
多品种快速切换交易；
依赖主观盘口感觉的交易；
需要专职盯盘才能执行的交易；
隔夜持仓策略；
加仓策略；
摊平策略；
网格策略；
重仓翻倍策略。
```

这些方向并不代表永远不能研究，但不属于当前阶段。

---

## 5. 禁止事项

本项目禁止：

```text
亏损摊平；
逆势加仓；
扛单回本；
无止损持仓；
亏损后扩大仓位；
为了提高胜率而扩大亏损；
把保证金够开仓当成可以交易；
用收益截图、冠军故事或单个案例反推策略规则。
```

如果一个候选方法依赖上述行为，则直接拒绝。

## 7. 当前结论

SmallFuturesLab 的研究起点不是信号，而是账户约束。

本项目宁可推进慢，也不能在账户边界、品种边界、周期边界和风险边界尚未清楚时进入策略实现。
