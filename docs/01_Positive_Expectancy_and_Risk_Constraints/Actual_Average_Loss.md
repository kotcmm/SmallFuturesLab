# 实际平均亏损

## 1. 定义

实际平均亏损是亏损交易的平均实际亏损金额。

```text
实际平均亏损 = 所有亏损交易的实际亏损金额之和 / 亏损交易笔数
```

它来自交易记录。

---

## 2. 示例

假设有 4 笔亏损交易：

| 亏损交易 | 实际亏损 |
|---:|---:|
| 第 1 笔 | 210 |
| 第 2 笔 | 230 |
| 第 3 笔 | 260 |
| 第 4 笔 | 220 |

则：

```text
实际平均亏损 = (210 + 230 + 260 + 220) / 4
实际平均亏损 = 230
```

---

## 3. 和 TradeR 的关系

`TradeR` 是单笔交易的计划风险。

```text
TradeR = |入场价 - 止损价| × 合约乘数 × 手数 + 预估交易成本
```

实际平均亏损和 `TradeR` 的比值，就是期望公式里的平均亏损倍数 `a`。

```text
a = 实际平均亏损 / TradeR
```

例如：

```text
TradeR = 220
实际平均亏损 = 230
```

则：

```text
a = 230 / 220
a = 1.045R
```

含义：

> 计划每笔最多亏 1R，但实际平均亏损是 1.045R。

---

## 4. 和 AccountR 的区别

`AccountR` 是账户允许的单笔风险上限。

```text
AccountR = 账户权益 × 单笔风险比例
```

`TradeR` 是某一笔交易的计划风险。

```text
TradeR <= AccountR
```

实际平均亏损是交易完成后的统计结果。

三者关系：

| 名称 | 含义 | 来源 |
|---|---|---|
| AccountR | 账户单笔风险上限 | 账户权益和单笔风险比例计算出来 |
| TradeR | 单笔计划风险 | 入场价、止损价、合约乘数、手数和成本计算出来 |
| 实际平均亏损 | 亏损交易的平均实际亏损金额 | 交易记录统计出来 |
| a | 实际平均亏损相对 TradeR 的倍数 | 实际平均亏损 / TradeR |

---

## 5. 使用方式

系统设计时，目标是：

```text
a <= 1
```

交易记录出来后，检查实际结果：

```text
实际 a = 实际平均亏损 / TradeR
```

如果长期出现：

```text
实际 a > 1
```

说明止损执行、滑点控制或成本估计存在问题。
